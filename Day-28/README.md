# Day 28 – Hospital Admission Readiness Simulator using Claude AI

## 🎯 Objective

The objective of this task was to build an interactive Hospital Admission Readiness Simulator using Claude AI.
The application simulates the hospital admission workflow, allowing users to evaluate patient readiness, manage prior authorization scenarios, reduce admission risks, and make informed admission decisions through an interactive interface.

---

## 🏥 Project Overview

The Hospital Admission Readiness Simulator is an educational web application that places the user in the role of a Hospital Admission Coordinator. 
It evaluates multiple admission factors such as prior authorization, insurance verification, documentation, physician orders, consent, and bed availability to determine whether a patient is ready for admission.
