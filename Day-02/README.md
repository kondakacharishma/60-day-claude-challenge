## 📚 Key Learnings

- Prompt Engineering is the skill of communicating effectively with AI.
- Adding context, role, and clear instructions improves output quality.
- Well-designed prompts reduce ambiguity and produce more accurate responses.
- Structured prompts save time and improve productivity.
