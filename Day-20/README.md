# 🚀 Day 20 – AI Face Puzzle Game

## 📌 Overview

As part of the **60-Day Claude Challenge (Day 20)**, I built an interactive **AI Face Puzzle Game** using Claude AI. The application captures a face through the webcam (or uses an uploaded image), converts it into puzzle pieces, and lets users solve the puzzle while tracking time and moves.

This project demonstrates how AI can generate complete browser applications using HTML, CSS, JavaScript, browser APIs, image processing, and responsive UI design.

---

## 🎯 Features

- 📷 Webcam access using the MediaDevices API
- 📸 Capture a live photo
- 🧩 Puzzle generation with 3×3, 4×4, and 5×5 difficulty levels
- 🔀 Automatic puzzle shuffling
- 🖱️ Drag-and-drop puzzle gameplay
- 📱 Touch support for mobile devices
- ⏱️ Live timer
- 🔢 Move counter
- ✅ Correct piece tracker
- 🏆 Win detection with results screen
- 💾 Leaderboard using Local Storage
- 🔄 Retake photo option
- 🎨 Modern responsive UI

---

## 🛠️ Technologies Used

- HTML5

## 🚀 How to Run

1. Download the project.
2. Open `face-puzzle-game.html` in a modern browser.
3. Allow camera permissions (or upload an image if supported).
4. Capture a photo.
5. Choose a difficulty level.
6. Solve the puzzle by arranging the pieces correctly.
7. View your completion time and leaderboard.

> **Note:** Webcam functionality works best over **HTTPS** or **localhost**.
