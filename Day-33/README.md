# Day 33 – Media Integrity Analyzer

## 📌 Challenge Overview

Today, I built an interactive **Media Integrity Analyzer** using **Claude AI**, **HTML**, **CSS**, and **JavaScript**. 
This application teaches media literacy by helping users identify misleading headlines, emotional manipulation techniques, and biased content through interactive challenges. The project demonstrates how AI can be used to create engaging educational tools that encourage critical thinking when consuming online media.

---

## 🎯 Objectives

- Learn the fundamentals of media literacy.
- Identify exaggerated or misleading headlines.
- Recognize emotional manipulation in online content.
- Analyze media credibility through interactive scenarios.
- Build a responsive, single-file HTML application with Claude AI.

---

## 🛠️ Features

- 🎨 Multiple color theme selection (including Claude Orange)
- 📰 Interactive Headline Detective challenge
- ❤️ Emotion Detector challenge
- 📊 Live Media Integrity metrics
- 📈 Final Media Integrity Dashboard
- 🔄 Replay with randomized scenarios
- 💻 Responsive offline HTML application
- ✨ Smooth animations and modern editorial-style interface

---

## 📚 What I Learned

- How sensational headlines use exaggerated language to attract attention.
- The importance of comparing headlines with the actual content.
- How emotional words can influence readers' decisions before facts are verified.
- Common media manipulation techniques such as fear, urgency, and vague authority.
- Practical strategies for evaluating online information critically.
- Improved my prompt engineering skills using Claude AI.
- Strengthened my frontend development skills using HTML, CSS, and JavaScript.
