# Day 26 – Prior Authorization Workflow Simulator using Claude AI

## 🎯 Objective

The objective of this task was to use Claude AI to build a fully interactive Prior Authorization Workflow Simulator as a single self-contained HTML application. The simulator demonstrates the U.S. healthcare prior authorization process through an engaging drag-and-drop workflow, helping users understand how authorization requests move between patients, providers, and payers.

---

## 🏥 Project Overview

The Prior Authorization Workflow Simulator provides an interactive learning experience where users guide a patient case through each stage of the authorization process. It explains the workflow, required documentation, payer review, and possible outcomes while tracking progress and efficiency.
