# Day 5 – Context Engineering

## Objective

Learn how providing context improves AI-generated outputs.

## Prompt A (Without Context)

Generated a generic 30-day learning roadmap.

## Prompt B (With Context)

Generated a personalized roadmap based on:

- Current Situation: B.Tech Student
- Current Skills: Python, Java, Cybersecurity Basics, HTML, CSS, GitHub
- Goal: Become an AI Engineer and Cybersecurity Professional
- Available Time: 3 Hours per Day
- Experience Level: Beginner
- Preferred Learning Style: Projects and Videos

## Comparison

### Which roadmap feels more personalized?

Prompt B

### Which roadmap would you actually follow?

Prompt B

### What role did context play?

Context helped Claude understand my background, goals, available time, and learning preferences, resulting in a more relevant and practical roadmap.

## Key Learnings

- Better context leads to better outputs.
- AI performs best when given relevant background information.
- Personalized prompts produce more actionable results.
