# Day 34 – Marketing Detective

## 📌 Challenge Overview

Today, I built an interactive **Marketing Detective** application using **Claude AI**, **HTML**, **CSS**, and **JavaScript**.
This detective-style learning experience transforms marketing analysis into an engaging investigation game where users solve fictional marketing campaign cases by examining evidence, organizing clues, and identifying the primary marketing mistake before reviewing an expert verdict.

---

## 🎯 Objectives

- Learn marketing strategy through interactive case investigations.
- Analyze campaign performance using real-world marketing metrics.
- Identify common marketing mistakes and their impact.
- Develop critical thinking and problem-solving skills.
- Build a responsive, single-file HTML application with Claude AI.

---

## 🛠️ Features

- 🎨 Multiple color themes (including Claude Orange)
- 🕵️ Interactive detective-style marketing investigations
- 📂 Randomized marketing campaign case files
- 📌 Drag-and-drop evidence board
- 📊 Campaign performance metrics analysis
- 💬 Customer feedback and social media insights
- ✅ Expert verdict with detailed explanations
- 📈 Marketing Learning Report
- 🔄 Replay with multiple randomized cases
- 💻 Responsive offline HTML application

---

## 📚 What I Learned

- How marketing metrics such as CTR, engagement, conversions, and sales help evaluate campaign performance.
- The importance of aligning campaign objectives, target audience, messaging, and marketing channels.
- How poor audience targeting or unclear messaging can reduce campaign effectiveness.
- The value of analyzing customer feedback and social media performance when improving marketing strategies.
- How interactive learning and AI can simplify complex marketing concepts.
- Improved my prompt engineering skills using Claude AI.
- Strengthened my frontend development skills using HTML, CSS, and JavaScript.
