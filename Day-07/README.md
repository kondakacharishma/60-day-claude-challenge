# Day 7 – Claude Model Selection & Reasoning Effort

## Objective

Learn how to choose the right Claude model and reasoning effort for different tasks.

## Questions Answered

- Current Situation: B.Tech Student
- Primary Activities: Coding, Learning, Cybersecurity, AI Research, Career Preparation
- Claude Usage: Daily
- Required Outputs: Learning Support, Coding Help, Deep Research

## My Claude Usage Strategy

### Recommended Primary Model

Sonnet

### Why?

It provides the best balance of intelligence, speed, and cost for my daily learning, coding, cybersecurity, and AI research tasks.

### When to Use Haiku

- Quick summaries
- Simple questions
- Fast rewrites
- Brainstorming

### When to Use Sonnet

- Coding
- AI learning
- Cybersecurity projects
- Resume improvements
- GitHub documentation

### When to Use Opus

- Deep research
- Career planning
- Complex project design
- Advanced reasoning

## Recommended Reasoning Effort

Low
- Simple tasks

Standard
- Daily learning and coding

High
- Debugging and cybersecurity analysis

Max
- Research papers and strategic planning

## Biggest Learning

Choosing the right Claude model is just as important as writing a good prompt. Using the correct reasoning effort improves both quality and efficiency.

## Key Takeaways

- Haiku is best for speed.
- Sonnet is ideal for most everyday tasks.
- Opus is best for complex reasoning.
- Select the reasoning effort based on task complexity.
