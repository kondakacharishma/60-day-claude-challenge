# Day 27 – Prior Authorization Story Simulator using Claude AI

## 🎯 Objective

The objective of this task was to build an interactive Prior Authorization Story Simulator using Claude AI. The application presents the U.S.
healthcare Prior Authorization process through a story-driven experience, helping users understand each stage using simple conversations between Rahul (patient) and Priya (healthcare operations specialist).

---

## 🏥 Project Overview

The Prior Authorization Story Simulator is an educational web application that explains how a prior authorization request progresses from diagnosis to insurance approval. 
The story is presented through interactive conversations, allowing users to make choices and learn about the healthcare workflow in an engaging and beginner-friendly way.

---

## ✨ Features Implemented

### 📖 Interactive Story

- 8 Story Chapters
- Rahul's Healthcare Journey
- Priya's Step-by-Step Explanations
- Interactive Choice-Based Dialogues
- Story Progress Bar
- Multiple Story Paths

---

### 🩺 Healthcare Workflow

- Doctor Visit
- Insurance Roadblock

### 💬 Interactive Learning
