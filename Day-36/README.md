# Day 36 – Cognitive Pattern Explorer

## 📌 Challenge Overview

Today, I built an interactive **Cognitive Pattern Explorer** using **Claude AI**, **HTML**, **CSS**, and **JavaScript**. 
This psychology-inspired self-reflection application helps users explore different thinking patterns through interactive scenarios, drag-and-drop activities, and personalized insights. 
The experience is designed to encourage reflection and self-awareness in a calm, engaging, and educational way without making any clinical assessments.

---

## 🎯 Objectives

- Explore different thinking tendencies through interactive scenarios.
- Reflect on decision-making styles in everyday situations.
- Learn how priorities and thought processes influence choices.
- Build a responsive, single-file HTML application that works completely offline.
- Create an engaging learning experience using vanilla JavaScript.

---

## 🛠️ Features

- 🎨 Calm and responsive user interface
- 🌿 Calm Mode and Stress Mode experiences
- 🧠 Chapter 1 – Discover Your Thinking Style
- 🎯 Chapter 2 – Choose Your Priorities (Drag-and-Drop)
- 🗺️ Chapter 3 – Map Your Thinking
- 📖 Personalized Reflection Journal
- 📊 Thinking Profile with percentage breakdown
- 📈 Progress tracking across all chapters
- 🔄 Replay with different modes
- 💻 Offline single-file HTML application

---

## 📚 What I Learned

- Learned how different situations influence decision-making and thinking patterns.
- Explored the characteristics of analytical, emotional, action-oriented, reflective, and overthinking styles.
- Understood the importance of balancing logical reasoning with emotional awareness.
- Improved my understanding of self-reflection through interactive learning experiences.
- Enhanced my prompt engineering skills using Claude AI.
- Strengthened my frontend development skills using HTML, CSS, and Vanilla JavaScript.
- Learned how interactive applications can make educational concepts more engaging and user-friendly.
