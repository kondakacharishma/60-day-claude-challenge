# Day 35 – 🧩 Prompt Puzzle: Master AI Prompting Through Play

## 📌 Challenge Overview

Today, I built an interactive **Prompt Puzzle** application using **Claude AI**, **HTML**, **CSS**, and **JavaScript**. 
This project teaches prompt engineering through fun, game-based challenges where users learn how to write effective AI prompts by building, improving, and selecting the best prompts for different scenarios.
The application provides instant feedback, live scoring, and a detailed performance report to help users strengthen their prompting skills.

---

## 🎯 Objectives

- Learn the fundamentals of prompt engineering.
- Practice creating clear and effective AI prompts.
- Compare weak prompts with optimized prompts.
- Understand how prompt quality affects AI-generated responses.
- Build an engaging, single-file HTML application that works completely offline.

---

## 🛠️ Features

- 🎨 Modern and responsive user interface
- 🧩 Three interactive prompt engineering challenges
- 🎯 Build the Prompt
- 🧹 Clean the Prompt
- ✅ Choose the Best Prompt
- 📊 Live scoring system with Accuracy, Time, Moves, Wrong Placements, Hints Used, and Optimization Bonus
- 📈 Prompt Performance Report
- 🧬 Prompt DNA Visualization
- 💡 Personalized feedback and learning recommendations
- 🔄 Replay with randomized scenarios
- 💻 Fully offline single-file HTML application

---

## 📚 What I Learned

- The importance of writing clear, specific, and structured prompts.
- How optimized prompts produce better AI-generated responses than vague prompts.
- The difference between weak, optimized, and over-engineered prompts.
- How prompt engineering principles improve communication with AI systems.
- The value of iterative prompt refinement for achieving accurate results.
- Improved my prompt engineering skills using Claude AI.
- Enhanced my frontend development skills using HTML, CSS, and JavaScript.
