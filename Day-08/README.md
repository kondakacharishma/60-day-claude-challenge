# Day 8 – Claude Artifacts: Personal Environmental Health Analyzer

## 🎯 Objective

Learned how to use **Claude Artifacts** to build an interactive AI-powered dashboard instead of generating plain text responses.

---

## 🚀 Project

**Personal Environmental Health Analyzer**

An interactive dashboard that analyzes environmental health using Air Quality Index (AQI) and water quality information. The application provides insights, charts, health impact analysis, and personalized recommendations.

---

## ✨ Features

- 📊 Interactive Dashboard
- 🌍 AQI Analysis
- 💧 Water Quality Analysis
- 📈 Charts & Visualizations
- 🎛 Interactive Filters
- 📋 City Health Cards
- 🚦 Environmental Health Score
- 💡 Personalized Recommendations
- 📱 Responsive Dark Theme UI
